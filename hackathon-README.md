# AI 文案工坊 (AI Copy Workshop)

> Build with Gemini XPRIZE Hackathon 2026 参赛项目  
> 用 AI 让小生意也能拥有专业文案团队

## 🎯 一句话介绍

一个由 Google Gemini API 驱动的 AI 文案生成工具，帮小商家、自媒体人、创业者 3 秒钟写出专业级商业文案。八大场景覆盖，零门槛使用。

## 💡 解决的问题

中国有超过 8000 万个体户和小微企业，他们每天都在为写文案发愁：
- 小红书笔记半天憋不出一篇
- 商品描述平淡如水转化率低
- 请不起专业文案（月薪 5000+）
- 用通用 ChatGPT 生成的文案「机器味」太重

AI 文案工坊用一个工具解决了所有这些问题。

## 🧠 AI-Native 设计

- **Gemini API 驱动**：每一项文案生成都是实时调用 Gemini 模型，不是套模板
- **场景化 Prompt 工程**：针对 8 种不同文案类型，精心设计了专用提示词
- **全自动工作流**：用户输入需求 → Gemini 生成 → 一键复制使用，零人工干预
- **AI 在「生产环境」运行**：不是一个 demo，是一个真正可用的商业产品

## 🛠️ 技术栈

| 层级 | 技术 |
|------|------|
| AI 模型 | Google Gemini 2.0 Flash (via Gemini API) |
| 云平台 | Google Cloud (Vertex AI / AI Studio) |
| 前端 | HTML5 + CSS3 + Vanilla JS (零框架依赖) |
| 存储 | localStorage (隐私优先) |
| 部署 | 单文件 HTML，双击即用，也可部署到 Cloud Run |

## 📊 商业模式

- **免费增值模式**：新用户 3 次免费试用
- **月度订阅**：¥29/月 (100次)
- **年度订阅**：¥199/年 (无限次)
- **目标客户**：电商店主、自媒体人、小微企业主、自由职业者
- **获客渠道**：小红书/闲鱼自然流量、口碑推荐
